# SM1-P.DERDERIAN
**Community Science Museum**

A responsive, multi-page website for a fictional science museum aimed at children aged 6-15 and their families.

Built as a vocational school projet for a front-end web development program.

**Description**

The Community Science Museum website is designed to inform visitors about the museum's exhibitions, events, services, and how to plan their visit. The design is child-friendly and accessible, with a focus on clarity and ease of navigation.

**Built With**

HTML5
CSS3
No JavaScript

**Features**

Mobile-first responsive design (mobile, tablet, desktop)
CSS-only burger menu with dropdown navigation
CSS-only search bar
Semantic HTML5 structure
WCAG accessibility considerations
W3C validated HTML and CSS

**Pages**

Home — Hero section, upcoming events, and featured exhibitions 
At the Museum — Services, accessibility info, and facilities 
Plan Your Visit — Opening hours, location, and visitor information 
Exhibitions — Overview of current exhibitions 
Contact Us — Contact form with confirmation page 
Coming Soon — Placeholder page with newsletter signup 
Confirmation — Form submission confirmation 
Newsletter Confirmation — Newsletter signup confirmation

**How to Run**

Open index.html in a browser, or visit the live site via GitHub Pages.
